# AFMall
A pure shopping App based on Kotlin + ARouter + MVP + RxJava2 + Retrofit + Dagger2 + 七牛云 + Glide

###### tips:

+ 本项目是用作学习的目的
+ IDE中提示缺少了Dagger开头的class直接编译即可
+ 发现bug或者更好的建议欢迎[issue](https://github.com/wangshufu/AFMall/issues)、email(1109341050@qq.com)
